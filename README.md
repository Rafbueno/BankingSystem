This project is a simple Banking System built in C# to practise Object-Oriented Programming (OOP) and unit testing using xUnit.
It models real-world banking concepts such as accounts, deposits, withdrawals, and transfers in a clear and beginner-friendly way.

Apply core OOP principles:
Encapsulation,
Abstraction,
Inheritance,
Polymorphism,
Build a modular console application,
Write and run unit tests with xUnit,
Separate application logic from test code.
